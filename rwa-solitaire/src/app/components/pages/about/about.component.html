About

This project was developed for the Razvoj Web Aplikacija course and presents a modern take on the classic Solitaire game. Built using modern web development technologies.

Key Features:

Built with Angular: Leveraging the power of a dynamic framework for a responsive and interactive user experience.

State Management with NgRx: Demonstrating efficient state handling for a seamless gameplay experience.

Reactive Programming with RxJS: Enhancing user interaction with powerful reactive streams.

Backend with NestJS: Providing a robust platform that tracks player statistics, manages leaderboards.
