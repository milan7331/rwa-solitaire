Welcome to RWA Solitaire!

 This project is a part of the Razvoj Web Aplikacija course and showcases a classic Solitaire game built entirely with modern web development technologies.

Key Features:

double_arrow Built with Angular Leveraging the power of a dynamic framework for a responsive and interactive user experience. double_arrow State Management with NgRx Demonstrating efficient state handling for a seamless gameplay experience. double_arrow Reactive Programming with RxJS Enhancing user interaction with powerful reactive streams. double_arrow Backend with NestJS Providing a robust platform that tracks player statistics, manages leaderboards.
